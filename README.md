## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements.

```bash
pip install requests
```
## How to use
1. Install python lastest version.
2. Download file in Github
3. Open cmd
4. Use Command 
``
pip install requests
``
5. Run 
``
python {file path}
``
